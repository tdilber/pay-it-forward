# Pay It Forward (Askıda Ekmek Kiosk Projesi)

Otomatikleştirilmiş halka temas eden ama güvenli bir bağış sistemi üzerine kafa yoruyordum. 
Aklıma Yeni bir fikir geldi. Daha önceki fikirlerim arasında tatmin edici hissiyattayım şuanlık. 
Goy goyu geçeyim: 

## Problem Tarifi
1- Güven problemi olmadan n kişi n kişiye bağışta bulunabilmeli. 
2- Halka temas edebilmeli, yani her köşede her mahallede olabilmeli. 
3- Kimlik bazlı limit olmalı. 
4- Temizlik hijyen problemi olmamalı. 
5- Veri güvenliği ve parasal güvenlik sağlamalı. 
6- Uzun süre Para tutmamalı. Para havuzu periyotları en

## Proje Tarifi
Kapalı Kutu Bir cihaz üretilecek (toplanacak). 

İçindeki bileşenler: 
1- Basit Barkod Makinesi 
2- Küçük bir ekran 
3- Kimlik okuyabilecek bir cihaz. (Ön arka iyi olur) 
4- Kamera
5- İnternete girebilecek ve cihazları çalıştıracak mininum donanıma sahip linux çalıştırabilecek bilgisayar.  (Raspberry Pi gibi)
